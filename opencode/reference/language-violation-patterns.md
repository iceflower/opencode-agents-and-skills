# Language Violation Patterns

> Pattern-based rules for detecting and correcting CJK (Chinese/Japanese) characters mixed in Korean sentences.
> Reference: §1 of AGENTS.md - CJK Mixed Language Prohibition

## Quick Reference

| Type | Unicode Range | Regex |
|------|---------------|-------|
| Chinese (Hanzi) | U+4E00–U+9FFF | `[\u4e00-\u9fff]` |
| Japanese Hiragana | U+3040–U+309F | `[\u3040-\u309f]` |
| Japanese Katakana | U+30A0–U+30FF | `[\u30a0-\u30ff]` |
| Korean (Hangul) | U+AC00–U+D7AF | **Allowed** |

## Chinese Pattern Rules

### Action Patterns (행동 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `让我{action}` | `{action}하겠습니다` | "让我修改" → "수정하겠습니다" |
| `让我直接{action}` | `직접 {action}하겠습니다` | "让我直接编辑" → "직접 편집하겠습니다" |
| `让我尝试{action}` | `{action} 시도하겠습니다` | "让我尝试另一种方式" → "다른 방법을 시도하겠습니다" |
| `让我检查{object}` | `{object} 확인하겠습니다` | "让我检查一下文件" → "파일을 확인하겠습니다" |
| `让我提供{object}` | `{object} 제공하겠습니다` | "让我提供确切的文本" → "정확한 텍스트를 제공하겠습니다" |
| `让我修复{object}` | `{object} 수정하겠습니다` | "让我修复这个问题" → "이 문제를 수정하겠습니다" |
| `让我添加{object}` | `{object} 추가하겠습니다` | "让我添加验证代码" → "검증 코드를 추가하겠습니다" |
| `让我继续{action}` | `{action} 계속하겠습니다` | "让我继续查找" → "계속 찾겠습니다" |
| `让我用{method}` | `{method} 사용하겠습니다` | "让我用最简单的方式" → "가장 간단한 방법을 사용하겠습니다" |

### State Patterns (상태 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `现在{state}` | `이제 {state}` | "现在我可以" → "이제 할 수 있습니다" |
| `现在我需要{action}` | `이제 {action}해야 합니다` | "现在我需要修改" → "이제 수정해야 합니다" |
| `看起来{observation}` | `{observation}한 것 같습니다` | "看起来有问题" → "문제가 있는 것 같습니다" |
| `首先{action}` | `먼저 {action}` | "首先检查" → "먼저 확인" |
| `然后{action}` | `그런 다음 {action}` | "然后进行编辑" → "그런 다음 편집" |

### Object Patterns (객체 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `{noun}限制` | `{noun} 제한` | "时间限制" → "시간 제한" |
| `{noun}功能` | `{noun} 기능` | "限制功能" → "제한 기능" |
| `{noun}内容` | `{noun} 내용` | "测试的内容" → "테스트 내용" |
| `{noun}问题` | `{noun} 문제` | "发现问题" → "문제 발견" |
| `{noun}错误` | `{noun} 오류` | "编译错误" → "컴파일 오류" |

### Sentence Patterns (문장 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `需要{action}` | `{action} 필요` | "需要详细分析" → "상세한 분석이 필요" |
| `必须{action}` | `{action}해야 합니다` | "必须马上处理" → "당장 처리해야 합니다" |
| `{verb}失败` | `{verb} 실패` | "加载失败" → "로드 실패" |
| `{verb}成功` | `{verb} 성공` | "编译成功" → "컴파일 성공" |

## Japanese Pattern Rules

### Verb Conjugation Patterns (동사 활용 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `{verb}して{action}` | `{verb}해서 {action}` | "検出して記録する" → "검출해서 기록하다" |
| `{verb}て{action}` | `{verb}어서 {action}` | "読んで理解する" → "읽어서 이해하다" |
| `{verb}後に{action}` | `{verb} 후에 {action}` | "確認後に実行" → "확인 후에 실행" |

### Prefix Patterns (접두사 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `再{verb}` | `재{verb}` 또는 `다시 {verb}` | "再起動" → "재시작" |
| `新{noun}` | `신규 {noun}` 또는 `새 {noun}` | "新機能" → "신규 기능" |
| `全{noun}` | `전체 {noun}` | "全ファイル" → "전체 파일" |

### Suffix Patterns (접미사 패턴)

| Pattern | Korean Transform | Example |
|---------|-----------------|---------|
| `{noun}的` | `{noun}적` | "基本的" → "기본적" |
| `{noun}化` | `{noun}화` | "自動化" → "자동화" |
| `{noun}性` | `{noun}성` | "重要性" → "중요성" |

## Usage Notes

### Pattern Matching Priority

1. **Longest match first**: `让我直接{action}` before `让我{action}`
2. **Context-aware**: Consider surrounding Korean text
3. **Exception handling**: Check `language-violation-exceptions.md` for special cases

### False Positive Prevention

Skip detection for:
- Code blocks (```...```)
- Import statements (`import`, `from ... import`)
- Comments (`//`, `#`, `/* */`)
- Package declarations
- URLs and file paths
- Proper nouns (names, brands)

### Integration with CJK Detector Plugin

This file is used by the `cjk-detector` plugin to:
1. Detect CJK characters in Korean sentences
2. Suggest corrections based on pattern rules
3. Log violations for review
