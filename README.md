# Agents.md

AI 코딩 에이전트를 위한 글로벌 규칙 파일(`AGENTS.md`)을 관리하는 저장소입니다.

## AGENTS.md란?

`AGENTS.md`는 AI 코딩 에이전트(OpenCode, Codex 등)가 읽어들이는 글로벌 행동 규칙 파일입니다. 에이전트의 언어, 보안, 문서 작성, 작업 완료 기준 등을 정의합니다.

## 포함된 규칙 (17개 섹션)

| #  | 섹션                       | 설명                                                     |
| -- | -------------------------- | -------------------------------------------------------- |
| 0  | 규칙 우선순위 및 충돌 해결 | 규칙 간 충돌 시 우선순위, 글로벌/프로젝트 규칙 우선순위  |
| 1  | 언어 규칙                  | 한국어 커뮤니케이션, 존댓말, 사고/추론 언어              |
| 2  | 정보 정확성                | 추측 금지, 외부 검증, 소스 충돌 해결, 보고 템플릿        |
| 3  | 프로젝트 위치 규칙         | 기본 경로 및 리소스 탐색 우선순위                        |
| 4  | 민감 정보 규칙             | API 키, 비밀번호 등 보안 가이드 및 긴급 대응             |
| 5  | 마크다운 문서 규칙         | 문서 포맷팅, 제목 계층, 테이블 스타일 표준               |
| 6  | 파일 수정 및 저장 규칙     | 편집 후 저장 일관성, markdownlint 검증                   |
| 7  | 대화 및 컨텍스트 규칙      | 맥락 파악, 문맥 판단, 다단계 작업 인식, 응답 범위 제한   |
| 8  | 태스크 완료 선언           | 완료 원칙, 사전 검증 항목, 보고 형식                     |
| 9  | 사용자 확인 프로토콜       | 위험 작업 시 확인 절차                                   |
| 10 | 파괴적 작업 안전           | 백업 프로토콜, 보호 파일 클래스, 실행 게이트             |
| 11 | 코드 품질 필수 원칙        | 6대 코드 품질 원칙, 테스트 행동 규칙, 에러 처리 원칙     |
| 12 | Skills 참조                | Agent Skills 표준, 로딩 방식, 사용 시점                  |
| 13 | 도구 사용 규칙             | 전용 도구 우선 사용, 병렬 실행, 외부 명령 안전           |
| 14 | 의존성 관리 규칙           | 추가/수정/제거 시 안전 규칙, 라이선스 확인               |
| 15 | Git 워크플로우 안전 규칙   | 커밋, 파괴적 명령, push 안전, 브랜치 관리                |
| 16 | 보안 코딩 실천             | OWASP Top 10 인식, 코드 수준 보안 (상세는 security 스킬) |

## 사용 방법

```bash
# AGENTS.md를 글로벌 설정 경로에 복사
cp AGENTS.md ~/.config/opencode/
```
